## Get started

1. Clone repository
   ```sh
   git clone https://github.com/QuestionsMark/prompt-creator-server.git
   ```
2. Install all packages
   ```sh
   npm install
   ```

3. Create mysql database

4. Create file /config/.env and fill all variables

5. Run nest app
    ```sh
    nest start --watch
    ```
